# Contributing to OpenUx

Thank you for your interest in OpenUx! We are excited to build the future of AI-driven UX together.

## How Can I Contribute?

### Reporting Bugs
- Use GitHub Issues to report bugs.
- Include steps to reproduce and system info.

### Suggesting Enhancements
- Open a GitHub Issue with the "enhancement" label.
- Describe the feature and why it would be useful.

### Pull Requests
1. Fork the repo.
2. Create a feature branch (`git checkout -b feature/amazing-feature`).
3. Commit your changes (`git commit -m 'Add amazing feature'`).
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request.

## Development Setup

### Monorepo Structure
- `web/`: Next.js frontend.
- `server/`: Node.js backend.

### Code Style
- Use TypeScript for everything.
- Follow the existing Prettier/ESLint configuration.
- Keep components focused and modular.

# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-24

### Added
- Initial OpenUx release.
- Landing page with AI prompt-focused hero section.
- Structured AI generation for UX workflows (page slugs, rules, actions, connections).
- Support for multiple AI providers (Google, OpenAI, Anthropic) via Vercel AI SDK.
- Workspace view for visualizing and iterating on AI-generated architectures.
- Monorepo setup with Next.js frontend and Node.js backend.
- Comprehensive open-source documentation (README, LICENSE, etc.).

### Fixed
- Improved model selection logic and Tier-based categorization.
- Refined input focus outlines and global dark theme styling.
- Resolved various TypeScript errors in the workspace generation loop.

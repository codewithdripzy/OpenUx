# GEMINI.md

## Project: AI Workflow Prompt System

This project is a **structured prompt system for AI-assisted product building**.

### Multimodal Reasoning Focus

When using Gemini:
- Focus on **system architecture** and **data flow**.
- Use Gemini's large context window to validate **cross-page dependencies**.
- Leverage multimodal capabilities for **UI/UX feedback**.

### Prompt Guidelines
- Keep inputs structured as JSON where possible.
- Use explicit role definitions for each page-level prompt.
- Ensure outputs match the expected state transitions defined in `workflow.json`.
